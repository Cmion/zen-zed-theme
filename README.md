# Zen Theme Experience for Zed Code Editor
Set of UI themes for Intellij IDEs:

- Zen Dark
- Zen Dark (Outlined)
- Zen Sakura (Cherry Blossom)
- Zen Sakura (Outlined)
- Zen Kaisen
- Zen Kaisen (Outlined)
- Zen Reverie
- Zen Reverie (Outlined)

These themes were designed to be easy on the eye, especially during long sessions.
